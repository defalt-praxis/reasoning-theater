The core insight: Models trained with reinforcement learning on chain-of-thought develop "performative reasoning" — they continue generating plausible-looking reasoning steps even after their internal representations have converged on an answer. On easy questions, the model's hidden states lock in the correct answer within the first ~20% of tokens. Probe-guided early exit can detect this and stop generation, saving massive compute with no accuracy loss. However, on genuinely hard questions, "aha moments" and backtracking correlate with real belief shifts — the reasoning is genuine.

Why This Matters

💰 Commercial implications:

• Inference cost reduction: Companies spending millions on reasoning model API calls could cut costs 30-80% by detecting when thinking is done
• Adaptive compute: Route easy questions through fast paths, save deep reasoning for hard ones
• Trust & transparency: Know when a model is genuinely reasoning vs. performing — critical for high-stakes applications
• Latency optimization: Real-time applications can exit early when confidence is high
